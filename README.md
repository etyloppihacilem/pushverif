# Pushverif

Use this script for your final check before pushing :)

`~/pushverif/pushverif.sh`

## Flags
- `-h`	displays this help
- `-p`	share (display and copy link to the repo
- `-r`	reward (open the github page so you can give me a star :))
- `-u`	update
- `-v`	verbose
